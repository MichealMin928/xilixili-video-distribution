## 2026-07-19｜每日运营技能与手工一键流程

- 新建个人 Codex 技能 `$xilixili-daily-ops`，固化内容证据检查、查重、排期、登录检查、四平台预填和失败处置流程。
- 新增 `npm run daily -- status|plan|start|publish`，由运营台本地 API 执行并复用同一专用 Chrome 会话。
- `start` 自动创建或复用当天任务，只补做尚未成功预填的平台，并停在发布按钮前。
- `publish` 按平台单独执行，阻止未预填、重复提交和早于排期的操作。
- 新任务记录来源清单和运营日期；旧任务通过标题与素材兼容查重。
- 用户决定暂不启用自动任务；未创建 Codex 定时任务、cron 或 macOS 后台服务。
